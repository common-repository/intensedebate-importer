Import comments from an IntenseDebate export file.

== Description ==

This plugin will import comments from an IntenseDebate export file. 

Please provide feedback and bug reports in the [Forums](http://wordpress.org/tags/intensedebate-importer?forum_id=10#postform)

== Installation ==

Upload the IntenseDebate Importer plugin to your blog, Activate it, then navigate to Tools -> Imports from your WordPress admin panel.
